=====================================================
 Organic Groups Global Limits
=====================================================
This module allows site administrators to impose
limits on the number of groups that users can create
and the number of members that can join said groups.

=====================================================
 Included Files
=====================================================
README.txt - this file
og_global_limits.info - the info file
og_global_limits.install - the install file
og_global_limits.module - the main module file
og_global_limits.admin.inc - the admin include file
